# Liquid Glass — модуль интерактивного эффекта «жидкое стекло»

Глобальный визуальный слой для кнопок, ссылок, аккордеонов, дропдаунов, модалок и плиток: матовое стекло с внутренним скосом, тонким бликом по краю, пастельными rim-подсветками и мягкой тенью. При нажатии — оптическая «вдавленность» и лёгкий сдвиг вниз.

## Быстрый старт

1. Подключите стили и скрипт:

```html
<link rel="stylesheet" href="liquid-glass.css">
<script src="liquid-glass.js"></script>
```

2. Инициализируйте (скрипт сам добавит общий SVG-фильтр в документ):

```javascript
LiquidGlass.init({
  root: document,
  selectors: 'button, [role="button"], .lg, [data-lg="auto"]', // или массив селекторов
  maxDisplacement: 24,
  pressDuration: 180,
  returnDuration: 300,
  useFilter: true
});
```

3. Разметьте элементы одним из способов:
   - Класс **`.lg`** — элемент участвует в эффекте.
   - Атрибут **`data-lg="auto"`** — включить на элементе.
   - Подсказка типа анимации (опционально): **`data-lg-type="press"`** | **`expand`** | **`toggle`** — пока обрабатывается единообразно, зарезервировано для тонкой настройки.
   - Для отключения: **`.lg-ignore`** или **`data-lg-ignore`**.
   - Полное отключение визуала: **`.no-lq`** / **`.no-lg`**.

По умолчанию модуль вешает обработчики на корень документа и обрабатывает клики по всем подходящим элементам (см. список селекторов в коде). Клавиатура: Space/Enter дают тот же эффект нажатия для фокусируемых элементов.

## API

| Метод | Описание |
|-------|----------|
| `LiquidGlass.init(opts)` | Инициализация с опциями (root, selectors, maxDisplacement, pressDuration, returnDuration, useFilter, reducedMode). |
| `LiquidGlass.press(element)` | Вручную включить состояние «нажато» на элементе. |
| `LiquidGlass.release(element)` | Снять состояние «нажато». |
| `LiquidGlass.pulseOpen(element)` | Импульс «раскрытие» (для открытия dropdown/accordion/modal). |
| `LiquidGlass.pulseClose(element)` | Импульс «закрытие». |
| `LiquidGlass.setReducedMode(true)` | Включить щадящий режим: меньше дисторции, можно отключить rim-эффекты через CSS. |
| `LiquidGlass.dispose()` | Снять все обработчики и удалить SVG-фильтр. |

### События expand/collapse

Для dropdown, аккордеона или модалки при открытии/закрытии вызывайте кастомные события с элемента, к которому применён эффект:

```javascript
element.dispatchEvent(new CustomEvent('lg:open', { bubbles: true }));  // при открытии
element.dispatchEvent(new CustomEvent('lg:close', { bubbles: true })); // при закрытии
```

Модуль подписан на эти события и запускает короткий импульс дисторции (растяжение/откат).

## CSS-переменные (настройка в :root)

| Переменная | Описание | Рекомендуемый диапазон |
|------------|----------|-------------------------|
| `--lg-button-radius` | Радиус скругления | 8–24px |
| `--lg-glass-bg` | Фон стекла | rgba(255,255,255,0.2–0.4) |
| `--lg-backdrop-blur` | Размытие фона | 8–20px |
| `--lg-displacement-max` | Макс. сила смещения фильтра (число для JS) | 12–32 |
| `--lg-press-duration` | Длительность нажатия, мс | 120–220 |
| `--lg-return-duration` | Длительность возврата, мс | 200–400 |
| `--lg-rim-left` | Левый/верхний rim (пастель) | rgba(233,215,255,0.4–0.6) |
| `--lg-rim-right` | Правый/нижний rim | rgba(255,214,224,0.4–0.6) |
| `--lg-shadow` | Тень под элементом | 0 8px 24px -6px rgba(0,0,0,0.1–0.2) |
| `--lg-inset-strength` | Внутренняя тень (объём) | 0 2px 6px rgba(255,255,255,0.4–0.6) |
| `--lg-press-y` | Смещение вниз при нажатии, px | 2–6 |
| `--lg-press-scale` | Масштаб при нажатии | 0.98–0.995 |

В JS при `init()` задаётся только `maxDisplacement` (соответствует силе `feDisplacementMap`). Остальные длительности можно дублировать в CSS-переменных для согласованности с анимациями.

## Советы по настройке

- **Усилить эффект нажатия:** увеличить `--lg-displacement-max` (и в `init({ maxDisplacement })`) и `--lg-inset-strength`.
- **Теплее/холоднее rim:** поменять `--lg-rim-left` / `--lg-rim-right` (например, больше розового или лавандового).
- **Быстрее отклик:** уменьшить `--lg-press-duration` (например, 120–150 ms).
- **Мобильные устройства:** оставить короткие длительности (150–200 ms), можно слегка уменьшить `maxDisplacement`, чтобы снизить нагрузку.
- **Десктоп:** можно увеличить blur и displacement для более «стеклянного» вида.

## Доступность и производительность

- Учитывается **prefers-reduced-motion**: при включённом снижении движения анимация дисторции отключается, остаётся только transform.
- Клавиатурный фокус: при нажатии Space/Enter к элементу применяется тот же визуал нажатия (`.lg--pressed`).
- Один общий SVG-фильтр в документе (`#lg-displace`), не по экземпляру на элемент.
- Обработчики повешены через делегирование на `root`; тяжёлые обновления только при активном нажатии (requestAnimationFrame для анимации scale фильтра).
- Для длинных списков достаточно, чтобы элементы имели класс `.lg` или входили в `selectors` — дополнительная разметка не нужна.

## Fallback

- Браузеры без `backdrop-filter`: в CSS включён fallback с полупрозрачным фоном и тенью.
- Класс **`.no-lq`** / **`.no-lg`** полностью снимает все стили Liquid Glass с элемента.

## Демо

Откройте `index.html` в браузере: кнопки, переключатель, dropdown, аккордеон, плитки навигации и модальное окно. При открытии/закрытии dropdown и аккордеона вызываются `lg:open` / `lg:close` для демонстрации импульса.

## Размер

- JS — ориентировочно до ~8–12 KB минифицированного.
- CSS — один файл, без внешних зависимостей.
