## 🎯 Новая логика работы

### Шаг 1: Пользователь запрашивает отчет
1. Нажимает "Получить отчет"
2. Выбирает диапазон дат (начало и конец)
3. Нажимает "Сгенерировать отчет"

### Шаг 2: Проверка лимита
- Система проверяет количество запросов за последние 7 дней
- Если >= 2 → показывает ошибку "Вы можете запрашивать отчет только 2 раза в неделю"
- Если < 2 → создает запрос

### Шаг 3: Создание запроса
В БД создается запись:
```sql
INSERT INTO reports_history (user_id, date_from, date_to, status, requested_at)
VALUES (1, '2026-01-01', '2026-01-31', 'pending', CURRENT_TIMESTAMP)
```

### Шаг 4: Показ сообщения
Всплывающее окно: **"Уже готовим ваш отчет! Он появится в течение 24 часов."**

### Шаг 5: Отображение в списке
Pending отчет показывается с:
- Оранжевым градиентом
- Анимированным спиннером
- Текстом "Отчет готовится... Появится в течение 24 часов"

## ⚠️ Что осталось сделать (отдельная задача)

Фоновая генерация отчетов НЕ реализована. Для завершения системы нужно:

1. **Создать background worker** (Python скрипт или cron job)
   - Раз в час проверяет pending отчеты
   - Вызывает `run_analysis()` для каждого
   - Обновляет status на completed/failed

2. **Добавить API endpoint** для ручного запуска worker
   - POST `/api/process-pending-reports` (admin only)

3. **Настроить cron** (Linux) или Task Scheduler (Windows)
   ```bash
   0 * * * * python /path/to/ml_part/report_worker.py
   ```

## 🧪 Как протестировать

### Тест 1: Создание отчета
1. Откройте `http://localhost:5173`
2. Войдите как Tima (user_id=1)
3. Перейдите в "Отчеты"
4. Нажмите "Получить отчет"
5. Выберите диапазон дат
6. Нажмите "Сгенерировать отчет"
7. **Должно появиться:** сообщение "Уже готовим ваш отчет..."
8. **В списке:** оранжевая карточка со спиннером

### Тест 2: Лимит (2 отчета в неделю)
1. Создайте 2 отчета
2. Попытайтесь создать 3-й
3. **Должна появиться ошибка:** "Вы можете запрашивать отчет только 2 раза в неделю"

### Тест 3: Просмотр pending отчета
1. Кликните на pending отчет (оранжевый)
2. **Должно:** ничего не происходить (нельзя открыть, пока не готов)
3. Или можно показать сообщение "Отчет еще готовится"
